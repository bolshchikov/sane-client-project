# Sane Client Project

## What's inside?

* TypeScript
* React
* Vite
* React Router
* Auth0
* Zustand
* Axios

## Getting started

1. Create a project using this Github template
2. `npm install`
3. Create `.env` file according to the example file and fill the values
4. `npm run dev`
5. Go to [http://localhost:5173](http://localhost:5173)
